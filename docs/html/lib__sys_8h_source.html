252// Generally speaking (the order does not really matter on this):
253// 1. Set the mode bits in the master control register (GRAPHICS + BITMAP), as well as the resolution.
254// 2. Load the color table for the bitmap into one of the graphics LUTs (start at $B4:2000 on the A2560U, $FEC8:2000 on the A2560K)
255// 3. Load the bitmap pixel data into video RAM (starts at $C0:0000 on the A2560U, $0080:0000 on the A2560K)
256// 4. Set the enable bit and LUT # in the bitmap control register for the layer you want
257// 5. Set the address of the pixel data in video RAM in the VRAM address pointer for the layer you want. This must be set relative to the beginning of VRAM, so if you're using the A2560U and the pixel data starts at $C2:0000, you'd store $02:0000 in the address register.

262// move.l #$0000000C,(a0) ; graphics mode & bitmap on.
263// move.l d0,$4(a0) ; zero border size/scroll/colour
264// move.l d0,$8(a0) ; clear border colour
