14/* about this class: Memory Manager
15 *
16 * Provides wrapper functions for the BGET memory manager, very slightly modified to have 2 separate pools, one for VRAM, one for normal RAM.
17 * All of the WORK of this code IS the BGET code.
18 * I made 2 very minor hacks:
19 * I removed the select best fit and auto memory manager functions to make it easier for (me) to read the code.
20 * I made it so that the "freelist" global property is not global, and is passed to the functions by my wrappers. This lets me have it manage separate pools for VRAM and normal RAM
21 *
22 *** things this class needs to be able to do
23 * Allocate and free memory in VRAM space
24 * Allocate and free memory in system RAM space
25 *
26 *
27 * STRETCH GOALS

29 *
30 * SUPER STRETCH GOALS
31 *
32 *
